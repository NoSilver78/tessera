# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its Code of Conduct.

In short: we pledge to make participation in Tessera a harassment-free experience
for everyone, and to act and interact in ways that contribute to an open,
welcoming, diverse, inclusive, and healthy community. Examples of unacceptable
behavior, the responsibilities of maintainers, scope, and enforcement guidelines
are defined by the Contributor Covenant text linked above.

## Reporting

Report unacceptable behavior privately to the maintainer via GitHub
(open a private security advisory, or contact the repository owner). All
complaints will be reviewed and investigated promptly and fairly. Maintainers
are obligated to respect the privacy and security of the reporter.
